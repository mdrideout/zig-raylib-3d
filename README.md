# Zig 3D Game

A 3D game built with Zig and Raylib.

## Requirements

- Zig 0.15.2 or later

## Building

```bash
zig build
```

## Running

```bash
zig build run
```

## Controls

- **Click** - Capture mouse for camera control
- **ESC** - Release mouse
- **W/S** - Move forward/backward
- **A/D** - Strafe left/right
- **E/Q** - Move up/down
- **Mouse** - Look around

## Testing

```bash
zig build test
```

## Dependencies

- [raylib-zig](https://github.com/raylib-zig/raylib-zig) - Zig bindings for Raylib
